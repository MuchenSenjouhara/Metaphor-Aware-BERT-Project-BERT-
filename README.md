# Metaphor-Aware-BERT-Project-BERT-
本项目基于 **BERT** 构建隐喻检测模型，并通过 **Gradio** 提供可交互网页界面，实现单句预测和批量文件预测。   This project builds a metaphor detection model based on **BERT** and provides an interactive web interface using **Gradio**, supporting both single-sentence prediction and batch file prediction.  ---
